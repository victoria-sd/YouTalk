# YouTalk Blog

Проект представляет собой веб-сайт блога, посвященный теме психологии и саморазвития. На сайте представлены статьи, информация о специалистах и услуги сервиса.

## Структура проекта

Проект включает в себя:
*   **Главную страницу:** С общим обзором, категориями и популярными статьями.
*   **Страницу статьи:** С подробным содержанием, изображениями, цитатами и боковой панелью с дополнительной информацией (например, о психологах, услугах).

## Как перейти на вторую страницу (статью)

Для перехода на страницу со статьей "Как не утонуть в тревоге и управлять своими страхами",найдите на главной странице ссылку с этим названием и кликните по ней.

## Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/victoria-sd/YouTalk.git
    ```

2.  **Перейдите в директорию проекта:**
    ```bash
    cd YouTalk
    ```

3.  **Откройте файл `index.html` в вашем браузере.**

## Зависимости

*   **Шрифты:** Подключены Google Fonts (`Montserrat`).
*   **Иконки:** Используются SVG-иконки.
*   **Изображения:** Изображения находятся в папке `images/`.

## Заметки по разработке

*   Проект сверстан с использованием семантических HTML-тегов.
*   Для стилизации использовались BEM-подобные классы.
*   Реализована адаптивная верстка для корректного отображения на различных устройствах.
